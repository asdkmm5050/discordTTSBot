# discordTTSBot

### Other language
- [中文](readme-zh.md)

### What is this?
- This is a Discord TTS bot, let it join a voice channel and  speak for you.

### Need
- discord
- discord_slash
- ffmpeg(like C:\fffmpeg\bin\fffmpeg.exe)

### How to use

- add a config.json file in ./
  ```
  {
  "token":"your token"
  }
  ```
- run python ./bot.py

### Commands
- `/join`
- `/talk "message"`
- `/leave `
- `/language "lang-code"`
