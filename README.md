# discordTTSBot

### NEED
- discord
- discord_slash
- fffmpeg(at C:\fffmpeg\bin\fffmpeg.exe)

### How to use

- add a config.json file in ./
  ```
  {
  "token":"your token"
  }
  ```
- run python ./bot.py

### Commands
- /join
- /talk "message"
- /leave 
- /language "language"
